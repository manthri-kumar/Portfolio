About the Project

The Hand Gesture Calculator is a touch-free calculator that interprets hand gestures to input digits and perform arithmetic operations. Using Python, OpenCV, and MediaPipe, it tracks hand and finger movements in real time for a seamless, interactive experience.

Technologies Used

  • Python
  • OpenCV
  • MediaPipe
  • HTML
  • CSS
